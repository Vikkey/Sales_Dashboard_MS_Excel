# Sales_Dashboard_MS_Excel
Developed a sales data visualization dashboard in Microsoft Excel using Pivot Tables, Pivot Charts, and Slicers to analyze sales performance effectively. Organized and transformed raw sales data to create interactive reports that allow users to filter and view metrics by individual sales representatives. Built dynamic dashboards to track sales trends, compare salesperson performance, and monitor key business metrics. Leveraged Excel's data analysis and visualization capabilities to generate actionable insights and support data-driven decision-making.

Key Features:
Created Pivot Tables to summarize and analyze sales data.
Built interactive dashboards using Pivot Charts and Slicers.
Enabled salesperson wise filtering for customized analysis.
Tracked sales performance, trends, and key metrics.
Improved reporting efficiency through dynamic visualizations.
Delivered actionable insights for business decision making.
